Which Bulbs Stay Lit?

There are 100 light bulbs in a row, all switched off. Then 100 people walk by, one at a time:

  • Person 1 flips every bulb (1, 2, 3, … 100).
  • Person 2 flips every 2nd bulb (2, 4, 6, …).
  • Person 3 flips every 3rd bulb (3, 6, 9, …).
  • … and so on, all the way to Person 100.

"Flip" means: if a bulb is off, turn it on; if it's on, turn it off. After everyone has walked by, which bulbs are still on? Make a guess first — then run it and watch.

Slow Fast
All 100 bulbs are off. Tap Next person to step through them yourself — after 8 people, Finish to end unlocks.
Why does this happen? (open after you've explored)

A bulb ends up on only if it is flipped an odd number of times. Bulb number k gets flipped once by each person whose number divides k. So bulb k is flipped once for every divisor of k.

Divisors normally come in pairs (for 12: 1×12, 2×6, 3×4 → six divisors, even → off). The only numbers with an odd number of divisors are perfect squares, because one divisor pairs with itself (for 16: 1×16, 2×8, 4×4 → five divisors).

So the survivors are exactly the perfect squares up to 100:

1, 4, 9, 16, 25, 36, 49, 64, 81, 100

That's 1², 2², 3², … 10² — exactly 10 bulbs left glowing.
